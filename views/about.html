How I Made This Website

Server

Built on NodeJs (with Express framework).
NodeJs is a javascript runtime, which allows to write server side code with javascript, and Express is a minimalistic javascript framework for Node.
Database used is MongoDB which is a NoSql database, and is integrated into express using mongoose package. The page views you see on the home page is updated on each visit using a collection on this database.
NodeJs Express MongoDB

Hosting

The site is hosted on Heroku, which is a cloud platform where apps can be launched very easily and also offers many addons for managing these apps.
The domain is registered on freenom.org
Also, the app is SSL/TLS encrypted using Cloudfare, along with other useful features from the Cloudfare platform like their own DNS servers, faster speeds and caching system.
Heroku Freenom Cloudfare

Frontend

Written with HTML,SCSS(a CSS preprocessor) and TypeScript
Animation on the front page uses Lottie, a json based animation format.
SCSS TypeScript Lottie

How everything connects

The devolopment is done on a local node server running on my PC, and the changes are pushed into github. The app hosted on Heroku looks for any changes in the github, and upon encountering any change, it does a new build sequence, which updates the app.
(Here, app = website)
